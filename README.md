# Responsive Landing Page

## Overview

This project is a responsive landing page developed as Task-01 during the Prodigy InfoTech Web Development Internship.

The webpage includes a fixed navigation bar that changes style while scrolling and provides smooth hover effects.

## Features

- Responsive Design
- Fixed Navigation Bar
- Scroll Animation
- Hover Effects
- Smooth UI

## Technologies Used

- HTML5
- CSS3
- JavaScript
